## 4. User-facing manual fallback: no-mistakes evidence prune
Delete secret GitHub gists recorded for a no-mistakes run.

The CI monitor normally deletes these gists automatically when it sees the PR merge or close. Use this command as a manual fallback for older runs, failed automatic cleanup, or monitors that are no longer running.

Warning: deleting these gists makes existing PR screenshots and video evidence links 404.

Usage:
  no-mistakes evidence prune [flags]

Flags:
  -h, --help         help for prune
      --pr string    PR number in the current repository whose recorded evidence gists should be deleted
      --run string   run ID whose recorded evidence gists should be deleted
